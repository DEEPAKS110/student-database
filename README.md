# Student Database Management System

Java desktop application (Swing) for managing student records with JSON persistence. A learning project demonstrating OOP, GUI development, and basic local data storage.

---

## Features

- Add / Update / Delete student records
- Search by ID or name
- Save and load records to/from JSON
- Simple dashboard / list view

---

## Technologies used

- Java
- Java Swing (GUI)
- Gson (JSON serialization)
- JSON file storage

---

## Quick start

1. Open the project in your Java IDE (IntelliJ IDEA, Eclipse).
2. Run `Main.java` or build and run the jar.
3. Data is saved/loaded from the included data/students.json.
