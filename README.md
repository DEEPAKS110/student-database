📊 Student Database Management System

A modern **Student Database Management System** developed using **Java** that enables users to efficiently manage student records through an intuitive graphical interface. The application supports CRUD operations, JSON data handling, search functionality, and report generation, making it suitable for academic institutions, student projects, and learning database concepts.



 📌 Project Overview

The **Student Database Management System** is designed to simplify the management of student information by providing an easy-to-use desktop application. Users can add, update, delete, search, and manage student records while maintaining data persistence through JSON storage.

This project demonstrates the implementation of Java programming concepts, file handling, data structures, object-oriented programming, and GUI development.



 ✨ Features

* ➕ Add New Student Records
* ✏️ Update Existing Student Information
* 🗑️ Delete Student Records
* 🔍 Search Students by ID or Name
* 📋 View Complete Student List

 💡 How It Works

1. Launch the application.
2. Add student information using the form.
3. Save records to the local JSON file.
4. Load saved records whenever needed.
5. Search, edit, or delete student details.
6. View all students through the dashboard.
